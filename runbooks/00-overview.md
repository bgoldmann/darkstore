# Darkstore runbooks

Operators must access the store **only via Tor** (Tor Browser or Tor-only path). No clearnet or VPN-only admin. See US-004, FR-4.

- **01-onion-deploy.md** – Deploy store as v3 onion service (US-001).
- **02-headers-logging.md** – Headers and minimal logging (US-002, US-003).
- **03-key-backup.md** – Encrypted .onion key backup and recovery (US-015).
- **04-incident-response.md** – Compromise and recovery (US-018).
- **05-escrow-disputes.md** – Escrow and disputes: platform PGP key, time limits, resolving disputes, no escrow/payment data in logs (US-020).
